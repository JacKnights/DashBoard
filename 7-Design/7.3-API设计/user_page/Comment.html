# GET /api/comments

获取所有评论信息

# Responses

接口

单条评论

interface Icomment {
    food_id: string; //标识种类
    food_name:string; //便于显示（不用再次发请求确定name）
    rating：number;
    content:string;
    //likes: number; //评论点赞数
    comment_time: string; //评论时间
}

评论列表

interface ICommentList{
    comments: Icomment[];
}

为评论点赞通过前端向后端Post一个Icomment对象实现。
